### Added
- Base justfile recipes (validate, lint, test, context)
- Language-specific recipes (Go, Python, Kubernetes)
- Planning skill with structured format
- Code review skill with checklist
- Commit message skill with conventional commits
- Justfile integration skill
- Prompt templates (plan, review-code, review-plan, commit)
- Pre-commit hook for validation
- Prepare-commit-msg hook for message generation
- Installation and uninstall scripts
- Example workflows
- Comprehensive README
- Testing checklist
- Contributing guide
